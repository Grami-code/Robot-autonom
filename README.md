📌 Descriere
Acest proiect reprezinta dezvoltarea unui robot mobil controlat wireless, capabil sa detecteze obstacole folosind un senzor infrarosu si sa reactioneze automat pentru evitarea coliziunilor.
Robotul combina componente hardware si software, realizand comunicarea intre microcontrolere prin WiFi si permitand control manual printr-un controller wireless.
Scopul proiectului a fost integrarea sistemelor embedded, electronicii si programarii pentru realizarea unui sistem robotic functional.

⚙️ Functionalitati
Control wireless al robotului prin controller; Detectare obstacole folosind senzor infrarosu; Oprire automata la apropierea de obiecte; Delay de siguranta dupa detectarea obstacolului;Comunicatie WiFi intre module; Control motoare prin driver dedicat

🧠 Tehnologii si componente utilizate
Hardware;
ESP32 (comunicatie WiFi si control principal);
Arduino UNO R4;
Senzor infrarosu pentru detectie obstacole;
Motor driver;
Motoare DC;
Controller wireless;
Sursa alimentare robot;
Software;
C / C++ (Arduino framework)
Programare embedded;
Comunicare WiFi intre microcontrolere;
Control motoare si citire senzori;
